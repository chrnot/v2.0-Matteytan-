# Matteytan

Ett digitalt klassrum för matematiklektioner. En interaktiv whiteboard med specialanpassade widgets för geometri, algebra, sannolikhet och taluppfattning.

## Funktioner

- **Interaktiv Whiteboard:** En fri yta för att placera och arrangera matematiska verktyg.
- **24+ Widgets:** Allt från tallinjer och bråkstavar till gradskivor och ekvationsvågar.
- **Ritverktyg:** Rita fritt på ytan över och under dina widgets.
- **Anpassningsbar Bakgrund:** Välj mellan rutnät, prickar, vit eller svart tavla.
- **Prestandaoptimerad:** Byggd med React och Vite, med lazy loading för snabb start.

## Kom igång

### Förutsättningar

Du behöver ha [Node.js](https://nodejs.org/) installerat på din dator.

### Installation

1. Klona repot:
   ```bash
   git clone https://github.com/ditt-anvandarnamn/matteytan.git
   cd matteytan
   ```

2. Installera beroenden:
   ```bash
   npm install
   ```

3. Starta utvecklingsservern:
   ```bash
   npm run dev
   ```

4. Öppna din webbläsare och gå till `http://localhost:5173` (eller den port som Vite anger).

## Bygg för produktion

För att skapa en optimerad produktionsbyggnad:

```bash
npm run build
```

Filerna hamnar i mappen `dist/` och kan publiceras på valfri webbserver (t.ex. Netlify, Vercel, GitHub Pages).
